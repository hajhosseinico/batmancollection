# What is this project?
A demo app which gets Batman collection from an API then shows the details.
# Main Features
1. Kotlin
2. MVVM
3. Hilt
4. Dagger
5. Live Data
6. Coroutines
7. Navigation Components
8. Room Persistence
9. Room caching
10. Retrofit2
11. Constraint Layout
12. Recycler view Using DiffUtil
13. Screen Rotation
14. Generic API response
